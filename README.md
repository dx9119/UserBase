# UserBase

Админ-панель для небольших проектов. Приложение построено на Spring Boot, Spring Security и Thymeleaf и подходит для случаев, когда нужно быстро развернуть управление пользователями без лишней сложности.

## Регистрация и вход

Поддерживается создание учётных записей, авторизация, выход из системы и смена пароля. Все операции проходят через Spring Security.

## Панель администратора

Администратор может просматривать список пользователей, искать их по имени или дате регистрации, изменять роли, блокировать и разблокировать аккаунты, удалять записи и смотреть историю изменений состояния пользователя.

## Профиль пользователя

Каждый пользователь видит свои данные, назначенные роли и информацию о текущей сессии.

## Локализация

В системе есть локализация на русский и английский языки. Переключение языка работает динамически, поэтому интерфейс можно менять прямо во время работы.

## Технологии

- **Backend:** Spring Boot 4.0.5, Spring Security, Spring Data JPA
- **Frontend:** Thymeleaf, Bootstrap 5
- **База данных:** PostgreSQL (по умолчанию), H2 для тестов
- **Сборка:** Maven

## Требования

- Java 21+
- PostgreSQL (или H2 для локального запуска)

## Установка

1. Клонировать репозиторий.

2. Настроить подключение к базе данных в `application.properties`:
   ```properties
   spring.datasource.url=jdbc:postgresql://localhost:5432/userbase
   spring.datasource.username=your_username
   spring.datasource.password=your_password
   ```

3. Запустить приложение:
   ```bash
   ./mvnw spring-boot:run
   ```

4. Открыть в браузере: http://localhost:8080
